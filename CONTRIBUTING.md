Contributing to Laravel Billing Core

First off, thank you for considering contributing! This project is a community effort, and we welcome all forms of contributions, from bug reports to feature requests and code patches.

How to Contribute

Reporting Bugs:

Use the GitHub Issues tab.

Provide a clear title and a detailed description of the steps to reproduce the bug.

Include which version of Laravel, PHP, and this package you are using.

Suggesting Enhancements:

Use the GitHub Issues tab with the "enhancement" label.

Explain the feature you'd like to see and why it would be useful.

Pull Requests (PRs):

Fork the repository and create a new branch from main.

Follow PSR-12: Ensure your code adheres to the PSR-12 coding standard.

Add Tests: Your PR must include tests for the new functionality.

Update the README: If you're adding a new feature, update the README.md to document it.

Run Tests: Before submitting, run the full test suite to ensure nothing is broken: composer test (You'll need to configure this in your composer.json scripts).

Thank you for your help!
